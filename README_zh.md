## 安装

### 使用 Lip

`lip install github.com/mrmagic2020/MSeries`

### 手动安装

1. 从 [releases page](https://github.com/mrmagic2020/MSeries/releases/latest) 下载最新版本。

2. 解压 **MSeries-windows-x64.zip**.

3. 将 **MSeries** 文件夹移至 BDS 根目录下的 **plugins** 文件夹。
